# Software Industry Analytics and Growth Trends Over the Years

## 📌 Overview
This project is a complete Data Visualization and Machine Learning analysis of the software industry from 2011 to 2025.

The project explores the transformation of software jobs across three major eras:

- 🟦 Golden Era (2011–2019)
- 🟥 COVID Era (2020–2021)
- 🟩 AI Era (2022–2025)

Using real-world developer survey datasets, this project analyzes salary trends, hiring demand, developer roles, automation risks, and the growing impact of Artificial Intelligence on the software industry.

# 📊 Dataset Information

The datasets used in this project were collected from:

- Kaggle
- Stack Overflow Developer Survey
- Public software industry datasets

The raw datasets contained large amounts of inconsistent and unstructured data.

### Data Processing Performed:
✅ Data Cleaning using Python Pandas  
✅ Missing value handling  
✅ Data transformation  
✅ Feature selection  
✅ Dataset merging  
✅ Salary normalization  
✅ Duplicate data removal  

After preprocessing and cleaning:

- 📌 ~48,000 rows were retained
- 📌 12 important columns were selected
- 📌 Cleaned datasets were prepared for dashboards and ML models
